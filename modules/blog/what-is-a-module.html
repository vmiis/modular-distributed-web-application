What is a module?

A module is a file (or group of files) that contains Html, JavaScript and CSS. It could be just a static page for showing information or it could be a data collection form. It can be hosted on any static web server and loaded into a content container when needed. Click here to see an example of a very simple module (a simple home page), and Click here to see the source con on CodePen. A module can also be a complete web page (with html, head and body) that will be loaded into a shadow-dom or an iFrame. A module should be a bit of independent so as it can be responsible and maintained by a individual person.
