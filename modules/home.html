I build this website for demonstrating the concept of modular distributed web application. I am looking for sponsors to keep my concept further developed.

My goal is to create an open source modular distributed cloud business operating system.

  • All modules should be open source and should be able to be hosted anywhere.
  • Business data and files should be stored in business owner's private database and storage.

I can build a web application with unlimited number of shared reusable modules distributed on different static web servers around the world and maintained by different people with different skills!

I am creating more various kind of shared reusable modules for fast building up a website and easy maintenance.
